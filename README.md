# 남창중 급식 알리미 도움말

## 유의사항
1. 이상한거 보내지 말아주세요

응답속도도 느려질 뿐만 아니라, 정확도에도 영향을 줍니다.

2. 좋지않은 메시지

보내시는 내용은 전부 제가 읽을 수 있습니다.

욕같은 언행은 삼가해주세요

3. 도배

1번과 같은 이유입니다.

## 사용방법

### 급식
급식은 2019.10.10과 같이 날짜 사이에 .을 넣어 입력해주세요


예시) 2019.10.10 급식알려줘


또는

"어제 급식알려줘"와 같이 어제 오늘 내일과 같은 말을 사용할 수 있습니다.
